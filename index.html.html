Bagian A · Membuat aplikasi di Claude (semua orang)

1

Buka website & daftar (gratis)

Buka claude.ai lewat browser (Chrome, Edge, atau Safari). Klik Daftar / Sign up, bisa pakai email atau akun Google. Gratis dan tanpa kartu kredit.

Tips: di HP pun bisa, ada aplikasi Claude di Play Store / App Store.

2

Tulis permintaan (prompt) sederhana

Di kolom chat, ketik permintaanmu. Makin jelas, makin bagus hasilnya. Pakai template di bawah ini — cukup ganti bagian yang disorot kuning, lalu tekan Enter.

Template prompt — tinggal isi & salin
Tolong buatkan aplikasi web sederhana untuk [tujuan aplikasi].

Pengguna bisa memasukkan [data yang diinput],
lalu aplikasi menampilkan [hasil yang diinginkan].

Buat tampilannya rapi dan mudah dipakai, dalam SATU file HTML,
dan pakai Bahasa Indonesia.

Contoh sudah diisi: "…aplikasi untuk menghitung OEE mesin. Pengguna memasukkan waktu operasi, downtime, jumlah produk, dan produk cacat, lalu aplikasi menampilkan nilai OEE dalam persen…"

3

Claude membuat aplikasinya

Dalam beberapa detik, Claude menampilkan aplikasi jadi di panel sebelah kanan (namanya Artifact). Kamu bisa langsung mencobanya di situ.

Belum pas? Tinggal bilang dengan bahasa biasa, contohnya:

  • "Tambahkan kolom untuk nama mesin."
  • "Ganti warna tombolnya jadi hijau."
  • "Tampilkan hasilnya dengan angka yang lebih besar."

Claude akan langsung memperbaikinya. Ulangi sampai cocok.

Cukup sampai sini kalau kamu di Jalur Santai. Aplikasimu sudah jadi dan bisa dipakai langsung di Claude. Lanjut ke Bagian B hanya kalau ingin menyimpan & mengedit kodenya sendiri.

Bagian B · Menyimpan & mengedit di VS Code (opsional)

4

Ambil / salin kodenya

Di panel Artifact, cari tampilan kode (atau menu titik-tiga ⋯ di pojok artifact). Di situ ada tombol Copy untuk menyalin seluruh kode, dan biasanya ada opsi Download untuk mengunduh filenya.

Kalau ragu, kamu juga bisa minta: "Tolong tampilkan kodenya supaya bisa saya salin."

5

Download & pasang Visual Studio Code

VS Code adalah editor kode gratis buatan Microsoft. Buka code.visualstudio.com → klik Download sesuai sistemmu (Windows / Mac / Linux) → jalankan file installer → ikuti Next → Next → Finish.

Di Windows, centang "Add to PATH" kalau muncul — biarkan opsi lainnya default.

6

Tempel kode & simpan sebagai file

  • Buka VS Code → menu File → New File.
  • Tempel kode tadi (Ctrl+V, atau Cmd+V di Mac).
  • Simpan: File → Save As, beri nama aplikasi.html — pastikan akhirannya .html.
7

Jalankan aplikasinya

Cara termudah: buka folder tempat menyimpan, lalu klik dua kali file aplikasi.html — langsung terbuka di browser. Selesai.

Cara lebih nyaman (otomatis refresh saat diedit): di VS Code, buka tab Extensions (ikon kotak di kiri), cari "Live Server", klik Install. Lalu klik kanan file .html → Open with Live Server. Setiap kali kamu simpan, tampilan browser ikut diperbarui.

8

Edit yang ringan-ringan

Di dalam kode, kamu bisa mengubah hal-hal sederhana tanpa takut rusak:

  • Teks & judul — tulisan di antara tanda > ... <.
  • Angka — misal target, batas, atau nilai default.
  • Warna — kode warna seperti #34D399 (hijau) atau #FFB627 (kuning).

Simpan (Ctrl+S), lihat hasilnya di browser. Salah? Tekan Ctrl+Z untuk membatalkan.

9

Kalau ada error — jangan panik

Di browser, tekan F12 → buka tab Console. Kalau ada tulisan merah, itu pesan error.

Cara paling gampang untuk pemula: salin pesan error itu, tempel ke Claude, lalu bilang: "Ada error ini, tolong perbaiki: [tempel pesannya]." Atau cukup jelaskan: "Tombolnya tidak jalan, kenapa ya?" — Claude akan membantu membetulkan dan memberi kode yang sudah diperbaiki.

Intinya: Claude adalah rekan codingmu. Tidak tahu cara memperbaiki sesuatu? Tanya saja dengan bahasa biasa.

Ide aplikasi untuk QCC & Kaizen

Tinggal masukkan ke template prompt di Langkah 2:

Kalkulator OEE
Hitung Availability × Performance × Quality dari data mesin.
Checklist Audit 5S
Centang area, skor otomatis, simpan hasil per shift.
Dashboard Defect Harian
Input jumlah cacat per kategori, muncul grafik Pareto.
Kalkulator Dampak Kaizen
Hitung waktu/biaya yang dihemat dari sebuah perbaikan.
Form Laporan QCC
Isi temuan & tindakan, hasilkan ringkasan rapi.
Pencatat 5-Why
Telusuri akar masalah langkah demi langkah.
3 tips pamungkas untuk pemula:
  • 1. Selalu minta "satu file HTML" — paling gampang dijalankan & dipindahkan.
  • 2. Mulai dari yang kecil & sederhana, lalu tambah fitur sedikit demi sedikit.
  • 3. Bingung apa pun? Tanya Claude dengan bahasa biasa. Tidak ada pertanyaan yang terlalu dasar.
